# 📊 Retail Sales Data Analysis

## 📌 Project Overview
This project analyzes a retail company's sales dataset to identify:
-  Top-performing categories  
-  Profitable regions  
-  Monthly sales trends  

The goal is to understand business performance and provide actionable recommendations.

---

## 🛠️ Tools Used
-  Python  
-  Pandas  
-  Matplotlib  

---

## 🔎 Analysis Performed
-  Sales vs Profit by Category  
-  Sales vs Profit by Region  
-  Monthly Sales Trend  

---

## 💡 Business Insights
-  Identified high-performing categories  
-  Analyzed regional revenue contribution  
-  Observed seasonal sales patterns  
-  Suggested focus areas for improving profitability  

---

## 🚀 Conclusion
Data-driven insights help businesses:
- Focus on high-profit segments  
- Improve underperforming regions  
- Plan inventory based on seasonal demand  

This project strengthened my data analysis, visualization, and business thinking skills.
