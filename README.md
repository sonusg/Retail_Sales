# Retail Sales Data Analysis

## Project Overview
This project analyzes a retail company's sales dataset to identify top-performing categories, profitable regions, and monthly sales trends.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Sales vs Profit by Category
- Sales vs Profit by Region
- Monthly Sales Trend

## Business Insights
- Identified high-performing categories
- Analyzed regional revenue contribution
- Observed seasonal sales patterns

## Conclusion
Data-driven insights can help businesses focus on profitable segments and improve overall performance.
